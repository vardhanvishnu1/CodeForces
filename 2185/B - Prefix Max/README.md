<div class="header"><div class="title">B. Prefix Max</div><div class="time-limit"><div class="property-title">time limit per test</div>1.5 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You are given an array of $$$n$$$ integers $$$a_1, a_2, \ldots, a_n$$$. </p><p>The value of an array is the sum of the maximums of each prefix of the array. More formally, the value of an array $$$a$$$ is $$$\sum_{i=1} ^{n} \operatorname{max}(a_1, \ldots, a_i)$$$. For example, the value of the array [$$$1, 2, 1$$$] is $$$\operatorname{max}(1) + \operatorname{max}(1, 2) + \operatorname{max}(1, 2, 1) = 1 + 2 + 2 = 5$$$.</p><p>You can choose two indices $$$i$$$ and $$$j$$$ and swap elements $$$a_i$$$ and $$$a_j$$$; this operation can be applied <span class="tex-font-style-bf">at most</span> one time.</p><p>Find the maximum possible value of the array $$$a$$$ after at most one operation.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line of the input contains a single integer $$$t$$$ ($$$1 \leq t \leq 100$$$) — the number of test cases.</p><p>The first line of each test case contains a single integer $$$n$$$ ($$$2 \le n \le 50$$$) — the length of the array $$$a$$$.</p><p>The second line contains $$$n$$$ integers $$$a_1, a_2, \ldots, a_n$$$ ($$$1 \le a_i \le 10^4$$$) — the array $$$a$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output the
