<div class="header"><div class="title">C. Removing Smallest Multiples</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You are given a set $$$S$$$, which contains the first $$$n$$$ positive integers: $$$1, 2, \ldots, n$$$.</p><p>You can perform the following operation on $$$S$$$ any number of times (possibly zero): </p><ul> <li> Choose a positive integer $$$k$$$ where $$$1 \le k \le n$$$, such that there exists a multiple of $$$k$$$ in $$$S$$$. Then, delete the <span class="tex-font-style-bf">smallest</span> multiple of $$$k$$$ from $$$S$$$. This operation requires a cost of $$$k$$$. </li></ul><p>You are given a set $$$T$$$, which is a subset of $$$S$$$. Find the minimum possible total cost of operations such that $$$S$$$ would be transformed into $$$T$$$. We can show that such a transformation is always possible.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line of the input contains a single integer $$$t$$$ ($$$1 \le t \le 10\,000$$$) — the number of test cases. The description of the test cases follows.</p><p>The first line contains a single positive integer $$$n$$$ ($$$1 \le n \le 10^6$$$).</p><p>The second line of each test case contains a binary string of length $$$n$$$, describing the set $$$T$$$. The $$$i$$$-th character of the string is '<span class="tex-font-style-tt">1</span>' if and only if $$$i$$$ is an element of $$$T$$$, and '<span class="tex-font-style-tt">0</span>' otherwise.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$10^6$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output one non-negative integer — the minimum possible total cost of operations such that $$$S$$$ would be transformed into $$$T$$$.</p></div><div class="sample-tests"><div
