<h3><a href="https://codeforces.com/contest/2188/problem/C" target="_blank" rel="noopener noreferrer">Restricted Sorting</a></h3>

<div class="header"><div class="title">C. Restricted Sorting</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><p>You are given an array $$$a$$$ of length $$$n$$$. For an integer $$$k$$$, we call it <span class="tex-font-style-it">piggy</span> if and only if it is possible to sort $$$a$$$ in non-descending order by performing the following operation an arbitrary number of times (possibly zero):</p><ul> <li> First, select two indices $$$i$$$ and $$$j$$$ ($$$1 \le i < j \le n$$$) such that $$$|a_i - a_j| \ge k$$$; </li><li> Then, swap $$$a_i$$$ and $$$a_j$$$. </li></ul><p>You need to determine the largest <span class="tex-font-style-it">piggy</span> integer $$$k$$$. If such an integer does not exist, output $$$-1$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1 \le n \le 2 \cdot 10^5$$$) — the length of $$$a$$$. </p><p>The second line contains $$$n$$$ integers $$$a_1,a_2,\ldots,a_n$$$ ($$$1 \le a_i \le 10^9$$$) — the elements of $$$a$$$.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2\cdot10^5$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single integer — the largest <span class="tex-font-style-it">piggy</span> integer $$$k$$$.</p><p>If such an integer does not exist, output $$$-1$$$ instead.</p></div><div class="sample-tests"><div
